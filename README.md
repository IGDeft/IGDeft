## 📂 Projetos em destaque

### 📦 Análise Estratégica de Logística — Olist

Análise exploratória utilizando **Python** para investigar como o custo do frete pode impactar as vendas da plataforma em diferentes regiões do Brasil.

**Principais análises**

- Concentração de vendas por estado e região.
- Relação entre frete e preço dos produtos.
- Comparação regional utilizando indicadores do dataset da Olist e dados demográficos do IBGE.
- Identificação de estados com potencial para expansão logística.

**Tecnologias:** Python • Pandas • Plotly

---

### 📈 Performance Comercial — Superstore

Análise de uma base comercial utilizando **PostgreSQL** para responder perguntas de negócio por meio de consultas SQL, agregações, CTEs e funções analíticas.

**Principais análises**

- Indicadores por categoria e subcategoria.
- Análise de lucro, vendas e descontos.
- Construção de consultas utilizando CTEs e funções analíticas.

**Tecnologias:** PostgreSQL • SQL
